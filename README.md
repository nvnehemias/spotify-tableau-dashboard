# Spotify Music Analytics Dashboard

## Project Overview

This project analyzes Spotify track data using Tableau Public to uncover insights into music popularity, genre trends, artist performance, and audio characteristics.

The dashboard was designed to simulate a real-world business intelligence solution by transforming raw music data into interactive visual analytics.

---

## Dashboard Features

- KPI cards for track and popularity metrics
- Genre popularity analysis
- Danceability vs popularity scatterplot
- Energy distribution histogram
- Top artist analysis
- Audio feature correlation analysis
- Interactive filters for genre, artist, and popularity

---

## Tools Used

- Tableau Public
- GitHub
- Python (optional data cleaning)
- Jupyter Notebook

---

## Key Insights

- Certain genres consistently outperform others in average popularity.
- Higher danceability often correlates with increased track popularity.
- High-energy tracks dominate many modern music genres.
- Some artists maintain consistently high popularity across multiple tracks.
